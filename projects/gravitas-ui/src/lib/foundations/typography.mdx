# Typography

Typography defines hierarchy, readability, and tone across Gravitas UI.

## Body text

This is standard body text. It should be easy to read and work well in forms, tables, cards, and documentation.

---

## Emphasis

**Bold text** is used for stronger emphasis.

_Italic text_ is used more sparingly.

`Inline code` can be used in docs and developer-facing content.

---

## Guidance

- Use headings to create clear hierarchy
- Avoid oversized text in dense interfaces
- Keep body text readable
- Use muted text only for secondary information
