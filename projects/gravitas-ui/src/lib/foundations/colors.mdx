# Colors

Gravitas uses a token-based color system so components stay consistent across the library.

---

## Core tokens

- `--gv-color-primary`
- `--gv-color-success`
- `--gv-color-warning`
- `--gv-color-danger`
- `--gv-color-info`

---

## Surface tokens

- `--gv-surface`
- `--gv-surface-muted`
- `--gv-surface-elevated`

---

## Text tokens

- `--gv-text-primary`
- `--gv-text-secondary`
- `--gv-text-muted`
- `--gv-text-inverse`

---

## Borders

- `--gv-border-color`
- `--gv-border-color-strong`

---

## Rule
